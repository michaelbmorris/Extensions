About

Extensions for built-in collection types. Methods are generic unless otherwise specified.

Types Extended

Array

Array extensions include the following:
Skip
Skips the specified count.

IEnumerable

IEnumerable extensions include the following:
ContainsIgnoreCase
Determines whether the IEnumerable<string> contains the specified string, ignoring case.
IsEmpty
Determines whether the specified IEnumerable is empty
